## lodash-math
matematiksel işmeler


```js
_.add(6, 4);
// => 10


_.ceil(4.006);
// => 5

_.ceil(6.004, 2);
// => 6.01

_.ceil(6040, -2);
// => 6100


//böler
_.divide(6, 4);
// => 1.5

_.floor(4.006);
// => 4

_.floor(0.046, 2);
// => 0.04

_.floor(4060, -2);
// => 4000

_.max([4, 2, 8, 6]);
// => 8


/** ortalama hesaplar */
_.mean([4, 2, 8, 6]);
// => 5


_.min([4, 2, 8, 6]);
// => 2


/* çarpar */
_.multiply(6, 4);
// => 24


_.round(4.006);
// => 4

_.subtract(6, 4);
// => 2


_.sum([4, 2, 8, 6]);
// => 20











```
